## What is Kronk Corp ?

Kronk Corp is a self-made organization to store some "side projects", especially libraries.

For now, it includes:
- Kronknet (a network library in C)
- Kronkworld (a header-only ECS in C++)
- Kronkflow (a task scheduler in C)
- Kronkargs (an argument parser in C)
- Kronklab (an unit test framework in C)
